# 🎵 Mini Music Player

A simple web-based music search application built using Python and Flask.
This project allows users to search for songs and listen to music previews directly in the browser.

The application fetches song data such as title, artist name, album artwork, and preview audio using the **iTunes Search API**.

---

## 🚀 Features

* 🔍 Search songs by name
* 🎧 Play music preview directly in the browser
* 🖼 Display album artwork
* 🎨 Dark themed user interface
* ⚡ Fast and simple music search
* 🌐 Web-based application (runs in browser)

## ⚙️ Installation Guide

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
```

### 2️⃣ Navigate to the Project Folder

```
cd YOUR_REPOSITORY_NAME
```

### 3️⃣ Install Required Libraries

```
pip install flask
pip install requests
```

### 4️⃣ Run the Application

```
python app.py
```

### 5️⃣ Open the Application

Open your browser and go to:

```
http://127.0.0.1:5000
```

## 📸 How the Application Works

1. User enters a song name.
2. The Flask backend sends a request to the iTunes Search API.
3. The API returns song information.
4. The app displays:

   * Song name
   * Artist name
   * Album artwork
   * Audio preview player

Users can listen to a short preview of the song directly from the webpage.
